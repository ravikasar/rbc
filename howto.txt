To run the app
---------------
step 1 : cd frontend && npm_install --This will install the required angular modules
step 2 : cd .. && mvn -f pom_jsp.xml spring-boot:run --This will start the springboot app and deploy the jsp
step 3 : cd frontend && ng serve --This will start angular frontend

To access in JSP mode
http://localhost:8080

To access in Typescript mode
http://localhost:4200
